## 에이전트는 모든 것을 혼자 하지 않는다

단순한 질문 답변이나 단일 파일 수정은 메인 에이전트가 직접 처리하는 것이 가장 빠릅니다. 하지만 **반복적인 일괄 작업(Batch Tasks)**이나 **방대한 아키텍처 탐색**은 메인 세션의 컨텍스트(대화 기록)를 불필요하게 낭비하게 만듭니다.

Gemini CLI의 진정한 힘은 작업을 전문 **서브 에이전트(Sub-agents)**에게 위임(Delegation)하고, **에이전트 스킬(Skills)**을 활용해 고도의 오케스트레이션을 수행하는 데 있습니다.

## 1. 내장 서브 에이전트 (Sub-agents)

에이전트는 각기 다른 전문성을 가진 하위 에이전트를 도구(Tool)처럼 호출할 수 있습니다. 위임된 작업은 메인 대화 이력을 깔끔하게 유지하면서도 압축된 요약 결과만 반환하므로, 비용과 속도 면에서 극강의 효율을 자랑합니다.

### Codebase Investigator (코드베이스 수사관)
저장소 전반의 아키텍처 매핑, 복잡한 시스템 의존성 파악, 깊이 숨겨진 버그의 근본 원인 분석에 특화되어 있습니다.
- **언제 쓰나요?**: "인증 흐름 전체를 추적해줘", "이 레포지토리의 라우팅 구조를 리포팅해줘" 같이 범위가 넓고 막연한 분석이 필요할 때.

### Generalist (제너럴리스트)
수많은 파일에 동일한 변경을 가하거나 출력이 매우 긴 셸 명령을 실행할 때 유용합니다. 메인 에이전트의 턴(Turn) 소모를 막아주는 최고의 방패막입니다.
- **언제 쓰나요?**: "src/ 폴더 하위의 모든 파일에 라이선스 헤더를 추가해줘", "린트 에러 50개를 전부 수정해줘" 같은 반복적 노동.

### CLI Help (도움말 전담)
Gemini CLI 자체의 기능, 단축키, 런타임 설정 등 시스템 사용법을 질문받았을 때 활용됩니다.

## 2. Agent Skills: 행동 강령 덮어쓰기

서브 에이전트가 "대신 일해주는 일꾼"이라면, **Skills**은 메인 에이전트의 "뇌를 전문가 모드로 포맷"하는 기능입니다.

`activate_skill` 도구를 통해 특정 이름의 스킬을 활성화하면, 에이전트는 기존의 일반적인 성향을 버리고 **해당 스킬에 정의된 전문가 절차(Instructions)**를 최우선으로 따르게 됩니다.

### 실무에서 자주 쓰이는 Built-in Skills
- `security-review`: 코드의 보안 취약점을 찾는 데만 집중하는 해커 마인드셋 장착.
- `code-review`: 엄격한 시니어 엔지니어 관점에서 리팩터링 제안과 리뷰 수행.
- `web-clone`: 제공된 URL을 분석하여 시각적/기능적으로 완벽한 클론 UI를 코딩해 내는 워크플로.
- `team`: 여러 에이전트가 tmux 기반으로 동시에 협업하는 다중 에이전트 모드.

### 나만의 스킬 만들기 (`skill-creator`)
"우리 팀만의 코드 리뷰 스타일"이나 "우리 회사 API 연동 표준"을 스킬로 만들고 싶다면 `skill-creator` 스킬을 활성화하세요. 에이전트가 사용자와 대화하며 새로운 `SKILL.md`를 작성하고 로컬에 설치해 줍니다. 이후부터는 복잡한 프롬프트 없이 단어 하나로 사내 표준 에이전트를 부를 수 있습니다.

## 3. 오케스트레이션 베스트 프랙티스 (전략적 위임)

Gemini CLI를 마스터하는 패턴은 다음과 같습니다.

1. **Plan Mode로 시작**: `/plan`을 입력하여 안전한 읽기 전용 샌드박스에서 구상을 시작합니다.
2. **조사 위임**: 아키텍처 파악이 막막하다면 `codebase_investigator`에게 "현재 프로젝트의 핵심 모듈을 파악해줘"라고 위임합니다.
3. **병렬 실행**: 서로 의존성이 없는 다수의 파일 검색이나 분석은 에이전트가 알아서 **병렬(Concurrent)**로 도구를 호출해 순식간에 끝냅니다.
4. **스킬 활성화 및 실행**: 조사 결과를 바탕으로 `/mode default`로 돌아온 뒤, 대규모 반복 리팩터링은 `generalist`에게 넘기거나 `activate_skill`로 특수 모드를 켜서 구현을 마무리합니다.

이러한 **전략적 오케스트레이션**이야말로 Gemini CLI가 일반 챗봇을 넘어선 진정한 '자율 엔지니어링 에이전트'로 작동하는 핵심 원리입니다.
